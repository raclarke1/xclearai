# XclearAI Brand Assets

## Logo Files

| File | Size | Use Case |
|------|------|----------|
| `logo.svg` | Vector | Web, scalable |
| `logo-white.svg` | Vector | Dark backgrounds |
| `logo-300.png` | 300px wide | Social media, small |
| `logo-600.png` | 600px wide | Email signatures, medium |
| `logo-1200.png` | 1200px wide | Print, presentations |

## Brand Colors

- **Primary Blue:** #1e40af
- **Accent Blue:** #3b82f6
- **Light Blue:** #93c5fd

## Typography

- **Font:** Inter (or system-ui fallback)
- **Logo weight:** 600 (Xclear) / 700 (AI)

## Usage

- Always maintain aspect ratio
- Minimum clear space: 1/4 logo height on all sides
- Don't stretch, rotate, or alter colors
